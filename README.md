## Clone Repo
```bash
cd ~
git clone https://github.com/zubxxr/sarsa-offloading.git
cd ~/sarsa-offloading
```

## Make virtual environment and install requirements
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Run code
```bash
cd ~/sarsa-offloading/v5
python3 main.py
```
